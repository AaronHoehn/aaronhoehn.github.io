Augmented Reality Prototype

I created an AR prototype that transforms a standard business card into an interactive digital hub. Using MindAR and A-Frame. This prototype is a web-based experience that tracks the card via a mobile camera to display dynamic content. The project was built using JavaScript and refined with HTML/CSS.
