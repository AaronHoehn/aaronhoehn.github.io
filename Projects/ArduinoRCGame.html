Arduino RC Game (2025)

Unity Arduino

Hand-Gesture Controller Prototype

In this prototype, I developed a Unity project where players drive an RC vehicle through a hilly landscape, collecting checkpoints along the way.

I built a glove controller: an Arduino-powered system using a gyroscope for tilt input and a flex sensor to detect lifting motions of the vehicle. To establish a wireless connection, I integrated a Bluetooth module that enables real-time communication with Unity.

Through this project, I was able to combine my knowledge of electronics with my game development skills and gained a deeper understanding of how to program microcontrollers.
