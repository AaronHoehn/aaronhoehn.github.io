Rock Paperplane Scissor

JavaScript p5.js Mobile Friendly Gameplay Prototype

This project was originally developed during my first semester in the Augmented and Virtual Reality degree program using Processing. It served as my introduction to the fundamentals of programming and game logic. To make the game more accessible and integrate it into my current portfolio, I later refactored and ported the entire project to JavaScript, utilizing the p5.js library. This transition allowed me to deepen my understanding of web-based game development and responsive design.
