Multiplayer Parkour-Controller Prototype (2025)

Unity Multiplayer Motion Capture OptiTrack

2nd Semester Solo Project

This project began as a test environment in which I developed a custom third-person parkour controller and integrated my own motion capture animations. The animations were recorded using the OptiTrack motion capture system.

Building on this foundation, I expanded the project into a multiplayer prototype in Unity. I successfully synchronized player movement and animations across the network using Photon PUN 2. This was my first multiplayer project and gave me valuable insight into networked gameplay systems, animation synchronization, and real-time player interaction.
