Rescur The Moon (2023)

Unity RPG

2nd Semester Solo Project

"Rescue the Moon" is a story-driven 3rd-person single-player RPG that I developed using Unity over a five-month period during my second semester at university. As the sole developer, I was responsible for every aspect of the project, from initial concept and design to final implementation.

This project significantly advanced my C# coding skills. I built my own complex systems, such as:

- 3rd Person Controller

- Enemy State Machine

- Vehicle Controls

- Linear Quest System
